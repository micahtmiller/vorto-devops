# pgadmin4

* Pull and run

```
docker pull dpage/pgadmin4:latest
docker run -p 80:80 \
    -e 'PGADMIN_DEFAULT_EMAIL=xxxx@gmail.com' \
    -e 'PGADMIN_DEFAULT_PASSWORD=xxxx' \
    -d dpage/pgadmin4
```

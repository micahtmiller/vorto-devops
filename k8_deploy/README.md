# Create new cluster manually

```
gcloud container clusters create helloworld-gke \
   --num-nodes 1 \
   --enable-basic-auth \
   --issue-client-certificate \
   --zone us-central1
```

# Get Cluster info

```sh
gcloud container clusters describe vorto-server-cluster --zone us-central1
```
